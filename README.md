# ConwaysGameOfLife

Just Conway's Game of Life.

## Build

```sh
$ make clean all
```

## Run

```sh
$ ./main.exe
```

## Runtime keyboard function

* `r` : restart the game randomly
* `f` : finish the game
* `d` : **DRUG MODE ON/OFF**
* `v` : verbose mode on/off
* `c` : circular board mode on/off
* `SPACE` : pause the game
* `+/-` : speed up/down
* `↑/↓/←/→` : resize the board
